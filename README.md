# adapt-timed  

**Timed** is a *presentation component* for the [Adapt framework](https://github.com/adaptlearning/adapt_framework).  

Its purpose is to allow content authors to set a delay before a next button appears to the learner.
## Installation
adapt install adapt-timed
## Limitations

No known limitations.
